## 🎯 Overview

The **Student Management System** is a lightweight, browser-based application that provides educational institutions with an intuitive platform to:

- **Create & Manage Student Records** – Store comprehensive student information with minimal effort
- **Track Academic Performance** – Monitor marks, grades, and subject-wise performance
- **Manage Attendance** – Track daily attendance and calculate attendance percentages
- **Financial Management** – Track fees status and scholarship eligibility
- **Data Analysis** – View real-time statistics and generate reports
- **Multi-Level Classification** – Organize students by class/grade levels with hierarchical views

### Key Highlights
- ✨ **100% Client-Side Processing** – No server required, all data stored in browser
- 🎨 **Modern UI/UX** – Clean, responsive design with smooth animations
- 📊 **Real-Time Analytics** – Instant statistics and breakdown reports
- 💾 **Persistent Storage** – Data saved in LocalStorage, survives page refreshes
- 🔍 **Search & Filter** – Quick access to student records with advanced filtering
- 📥 **CSV Export** – Export student data for external analysis

## ✨ Features


#### 1. **Student Management**
   - Add, edit, and delete student records
   - Store comprehensive student information:
     - Personal details (name, roll number, email, address)
     - Academic information (marks, grades, subjects, projects, future goals)
     - Attendance tracking (percentage-based)
     - Financial status (fees paid/pending)
     - Health information (blood group, medical conditions)
     - Personal interests (hobbies)
     - Parent/Guardian contact information

#### 2. **Dashboard Statistics**
   - **Total Students** – Quick view of enrolled students (clickable for breakdown)
   - **Average Marks** – Overall academic performance percentage
   - **Scholarships** – Count of scholarship-eligible students
   - **Attendance** – Average attendance percentage across all students
   - **Fees Paid** – Quick reference for fee collection

#### 3. **Search & Filtering**
   - Real-time search by student name or roll number
   - Filter students by class/grade level
   - Combined search + filter functionality
   - Instant result updates as you type

#### 4. **Total Students Breakdown**
   - View student distribution across educational levels:
     - **Pre-Primary** (Nursery, LKG, UKG)
     - **Primary** (Class 1-5)
     - **Lower Secondary** (Class 6-8)
     - **Secondary** (Class 9-10)
     - **Higher Secondary** (Class 11-12)
   - Gender-based student counts (Boys/Girls)
   - Visual grade cards with progress bars
   - Tabular breakdown with room assignments

#### 5. **Data Management**
   - Edit student records anytime
   - Delete student records with confirmation
   - View all records in organized table format
   - Automatic data persistence using browser LocalStorage

#### 6. **Export & Reporting**
   - Export all student data to CSV format
   - Compatible with Excel, Google Sheets, and other spreadsheet applications
